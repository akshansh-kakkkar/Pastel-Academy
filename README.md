# Pastel Academy
It's a product information checker platform with some extra features which I have implemented for practice 

## Features
- Message fetching from backend
- Custom loading screen
- Custom 404 page
- fake items from "https://fakestoreapi.com/products"
- fake discussion from "https://jsonplaceholder.typicode.com/posts"
- Client Side Rendering (CSR) powered by SWR for seamless lightweight data etching.

## Tech Stack
- Next.js
- TailwindCSS
- Framer Motion
- SWR

## Installation

### Clone the repository
```
git clone https://github.com/akshansh-kakkkar/Pastel-Academy.git
```

### Install Dependencies
```
bun install
```
or 
```
npm install
```

### Run the Project
```bash
bun dev
```
or

```bash
npm run dev
```
